# Pixel Scene Assistant Mock Demo

## What it is
A clickable mock demo for Step 2 validation.

## What you can try
- input a scene brief
- click `生成 mock 草稿`
- continue with `变体` / `refine`
- view structured Scene Spec
- export Scene Spec JSON
- export full Scene Package (spec + assets + metadata)

## What it now represents
This is no longer just a UI mock. It's a **hybrid MVP skeleton** with real machine-readable output structure aligned to production workflow needs.

## What it proves
This demo validates the interaction direction of `Pixel Scene Assistant` as an AI-assisted pixel scene creation capability.

## What it does NOT prove
- no real generation backend yet
- no real image synthesis yet
- no production workflow integration yet

## Open locally
Open `index.html` in browser.
